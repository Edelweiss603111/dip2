Название
Название точки обслуживания. Обязательное поле.
Время регистрации
Время регистрации устройства в системе. Заполняется автоматически.
Авторизация
Отображает, авторизовано ли устройство в системе. Авторизацию можно отозвать в любой момент.
Тип
Определяет тип точки обслуживания. Тип бывает двух видов:
  • мобильный терминал - приложение для мобильных устройств на базе Android, эмулирующее работу турникета
  • контроллер - валидатор, установленный возле объектов контроля
Тип контроллера
Тип контроллера: Arduino или CardLan.Только для точки обслуживания типа "контроллер".
Период обновления
Периодичность, с которой контроллер будет посылать запрос к серверу для получения тарифов и настроек. Измеряется в минутах. Только для точки обслуживания типа "контроллер".
Временная зона
Временная зона, в которой фактически расположено устройство. Только для точки обслуживания типа "контроллер".
Проверка прохода онлайн
Разрешение или запрет для валидатора на проверку правомерности прохода посетителя через данную точку напрямую с сервера, без считывания записанных на карту услуг. Только для точки обслуживания типа "контроллер".
Последнее получение настроек
Показывает, когда в последний раз точка обслуживания загружала настройки с сервера. Только для точки обслуживания типа "контроллер".
Последнее получение тарифов
Показывает, когда в последний раз точка обслуживания загружала тарифы с сервера. Только для точки обслуживания типа "контроллер".
Последняя загрузка на сервер общего лога
Показывает, когда в последний раз происходила загрузка на сервер общего лога устройства. Только для точки обслуживания типа "контроллер".
Последняя загрузка на сервер лога проходов
Показывает, когда в последний раз происходила загрузка на сервер лога проходов. Только для точки обслуживания типа "контроллер".
Последняя загрузка на сервер лога транзакций
Показывает, когда в последний раз происходила загрузка на сервер лога транзакций. Только для точки обслуживания типа "контроллер".
